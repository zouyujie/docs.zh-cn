# <a name="help--visual-basic"></a>/help，/? (Visual Basic)
显示编译器选项。  
  
## <a name="syntax"></a>语法  
  
```  
/help  
' -or-  
/?  
```  
  
## <a name="remarks"></a>备注  
 如果在编译中包括此选项，不创建任何输出文件并不进行编译。  
  
> [!NOTE]
>  `/help`选项不是可从[!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]开发环境中; 只有当从命令行进行编译，它才可用。  
  
## <a name="example"></a>示例  
 下面的代码显示从命令行帮助。  
  
```  
vbc /help  
```
