# &lt;type1&gt;“&lt;typename1&gt;”与为&lt;type2&gt;“&lt;typename2&gt;”中的“&lt;eventname&gt;”事件隐式声明的成员冲突
类型成员名称与为事件隐式创建的成员名称冲突。 事件隐式创建多个隐式变量。 例如，声明 `Event X` 隐式声明了名称 `XEventHandler`、`XEvent`、`add_X` 和 `remove_X`。  
  
 **错误 ID：**BC31061  
  
### 更正此错误  
  
-   重命名显式声明的成员，以消除命名冲突。
