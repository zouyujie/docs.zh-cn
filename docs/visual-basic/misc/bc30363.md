# &quot;Sub New&quot; 不能在接口中声明
你试图在接口中声明 `Sub New`。 由于它是构造函数，因此在创建类时，`Sub New` 只能运行一次。 不能从同一个类或某个派生类中另一构造函数的首行代码以外的任何位置对它进行显式调用。  
  
 **错误 ID：**BC30363  
  
### 更正此错误  
  
-   删除 `Sub New` 或将其移到代码中更适当的位置。
