# “AddHandler”和“RemoveHandler”方法参数不能声明为“ByRef”
不能使用 `ByRef` 修饰符声明 `AddHandler` 和 `RemoveHandler` 方法的参数。  
  
 **错误 ID：**BC31134  
  
### 更正此错误  
  
-   从参数中删除 `ByRef` 关键字。
