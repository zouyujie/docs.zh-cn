# XML 标记需要有结束的“&gt;”
XML 文本的 XML 标记缺少必需的结束字符“\>”。  
  
 **错误 ID：**BC31166  
  
### 更正此错误  
  
-   将结束字符“\>”添加到 XML 文本的 XML 标记的结束位置。
