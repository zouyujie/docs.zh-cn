# /win32manifest 选项已忽略
已忽略 \/win32manifest 选项。 只有在目标是程序集时才能指定该选项。  
  
 当 `/target` 选项设置为 `module` 时，已向 Visual Basic 编译器传递 `/win32manifest` 编译器选项。  
  
 **错误 ID：**BC2034  
  
### 更正此错误  
  
1.  删除 `/win32manifest` 编译器选项或将 `/target` 选项设为 `exe`、`winexe` 或 `library`。
