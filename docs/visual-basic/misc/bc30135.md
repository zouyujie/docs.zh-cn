# &quot;SyncLock&quot; 语句在即时窗口中无效
`SyncLock` 语句会同步线程，不允许用于调试上下文中。  
  
 **错误 ID：**BC30135  
  
### 更正此错误  
  
-   不要在“即时”窗口中发出 `SyncLock` 语句。
