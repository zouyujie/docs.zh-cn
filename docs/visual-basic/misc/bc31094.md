# 无法找到接口 &lt;interfacename&gt; 的实现类“&lt;classname&gt;”
互操作程序集中的实现类不可用。  
  
 **错误 ID：**BC31094  
  
### 更正此错误  
  
1.  验证 COM 对象的类型库是否有效。  
  
2.  使用类型库导入程序 \(Tlbimp.exe\) 来手动创建互操作程序集，然后再从你的项目添加对该互操作程序集的引用。
