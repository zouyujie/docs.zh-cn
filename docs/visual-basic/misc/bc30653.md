# 需要对模块“&lt;modulename&gt;”（包含类型“&lt;membername&gt;”）的引用
需要对模块“\<modulename\>”（包含类型“\<membername\>”）的引用。 请向项目中添加一个。  
  
 项目包含一个类型，该类型依赖于项目引用中未包括的程序集中的另一个类型。  
  
 **错误 ID：**BC30653  
  
### 更正此错误  
  
-   在项目中添加依赖类型的项目引用。
