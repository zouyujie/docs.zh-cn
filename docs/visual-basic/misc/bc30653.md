# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-type-39ltmembernamegt39"></a>需要对模块引用&lt;modulename&gt;包含的类型'&lt;membername&gt;
需要对模块引用\<modulename&1;> 包含的类型 '\<membername&1;>。 请向项目中添加一个。  
  
 项目包含一个类型，该类型依赖于项目引用中未包括的程序集中的另一个类型。  
  
 **错误 ID：** BC30653  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在项目中添加依赖类型的项目引用。
