# &quot;Interface&quot; 必须以匹配的 &quot;End Interface&quot; 结束
出现 `Interface` 语句而没有相应的 `End Interface` 语句。 必须使用 `End Interface` 语句结束接口定义。  
  
 **错误 ID：**BC30253  
  
### 更正此错误  
  
1.  检查 `Interface` 成员的格式是否正确。  
  
2.  在接口定义的末尾添加一个 `End Interface` 语句。
