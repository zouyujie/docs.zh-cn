# &lt;specifier1&gt; &lt;type&gt; 不能继承自 &lt;specifier2&gt; &lt;type&gt;，因为它将扩展基 &lt;type&gt;的访问权限
你尝试将继承自基类的公共类标记为 `Private` 或 `Friend`。  
  
 **错误 ID：**BC30509  
  
### 更正此错误  
  
-   将基类声明为 `Public`，或者将继承的类声明 `Private` 或 `Friend`。
