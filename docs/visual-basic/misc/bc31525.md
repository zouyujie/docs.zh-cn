# <a name="type-39lttypenamegt39-is-imported-from-different-versions-of-assembly-39ltassemblynamegt39"></a>类型 '&lt;typename&gt;导入来自不同版本的程序集'&lt;assemblyname&gt;
类型 '\<typename&1;> 来自不同版本的程序集导入\<程序集名称&1;>。 同一程序集的不同版本不能用于同一项目。  
  
 从同一程序集的不同版本导入了类型，产生了多义性。  
  
 **错误 ID：** BC31525  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除对同一程序集的多个版本的引用。
