# 类型“&lt;typename&gt;”是从程序集“&lt;assemblyname&gt;”的不同版本导入的
类型“\<typename\>”是从程序集“\<assemblyname\>”的不同版本导入的。 同一程序集的不同版本不能用于同一项目。  
  
 从同一程序集的不同版本导入了类型，产生了多义性。  
  
 **错误 ID：**BC31525  
  
### 更正此错误  
  
-   删除对同一程序集的多个版本的引用。
