# “Implements”在模块中无效。
`Implements` 语句或子句发生在模块内。 模块不能实现接口。  
  
 **错误 ID：**BC30231  
  
### 更正此错误  
  
-   删除 `Implements` 语句或子句或重新输入模块作为类。
