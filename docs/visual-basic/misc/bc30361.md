# “&lt;property1&gt;”和“&lt;property2&gt;”中只有一个声明为“Default”，因此它们无法相互重载
如果某个属性指定 `Default`，则在该名称上重载的所有属性也必须指定 `Default`。  
  
 **错误 ID：**BC30361  
  
### 更正此错误  
  
-   请确保所有重载的属性均声明为 `Default`。
