# 无法找到编译此结构必需的“&lt;name&gt;”
无法找到编译必需的元素。  
  
 **错误 ID：**BC30930  
  
### 更正此错误  
  
1.  提供必要的帮助程序元素。  
  
2.  确保代码是有效的托管代码。
