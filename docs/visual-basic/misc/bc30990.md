# 无法在对象初始值设定项表达式中初始化成员“&lt;membername&gt;”，原因它不是字段或属性
对象初始值设定项列表不能包含共享成员、常量或方法调用。 初始值设定项列表中的成员必须是字段或属性，并且属性成员不可要求提供参数。  
  
 **错误 ID：**BC30990  
  
### 更正此错误  
  
-   从对象初始值设定项列表中删除所有共享成员、常量、方法调用或具有参数的属性。
