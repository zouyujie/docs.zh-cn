# “NotInheritable”类不能将成员声明为“&lt;specifiername&gt;”
`NotInheritable` 类不能使用重写修饰符，因为其成员不能被重写。  
  
 **错误 ID：**BC30607  
  
### 更正此错误  
  
-   从类定义中删除重写修饰符，如 `Overridable``NotOverridable` 或 `MustOverride`。
