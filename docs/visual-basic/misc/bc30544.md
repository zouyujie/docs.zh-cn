# 方法不能同时包含“Try”语句和“On Error”或“Resume”语句
你已将 `Try` 语句和 `On Error` 或 `Resume` 合并。  
  
 **错误 ID：**BC30544  
  
### 更正此错误  
  
1.  删除 `On Error` 或 `Resume`。
