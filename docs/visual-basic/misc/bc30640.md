# &quot;Option Explicit&quot; 的后面只能跟 &quot;On&quot; 或 &quot;Off&quot;
`Option Explicit` 语句必须指定 `On` 或 `Off`。 其他所有值均无效。  
  
 **错误 ID：**BC30640  
  
### 更正此错误  
  
-   指定 `Option Explicit On` 或 `Option Explicit Off`。
