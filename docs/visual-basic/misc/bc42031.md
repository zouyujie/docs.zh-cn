# 永远不会到达“Catch”块；&lt;exception&gt; 已在上面同一个“Try”语句中处理
无法达到代码中的 `Catch` 块，因为它已经在前面的 `Try` 块中得到处理。  
  
 默认情况下，此消息是一个警告。 有关隐藏警告或将警告视为错误的详细信息，请参阅 [在 Visual Basic 中配置警告](/visual-studio/ide/configuring-warnings-in-visual-basic)  
  
 **错误 ID：**BC42031  
  
### 更正此错误  
  
-   删除多余的语句。
