# “RaiseEvent”声明必须以匹配的“End RaiseEvent”结束
`RaiseEvent` 声明必须以 `End RaiseEvent` 语句结束。  
  
 **错误 ID：**BC31117  
  
### 更正此错误  
  
-   检查 `RaiseEvent` 声明是否以 `End RaiseEvent` 语句结束。
