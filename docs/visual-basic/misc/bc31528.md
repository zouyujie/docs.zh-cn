# 构造函数间接引用包含“&lt;typename&gt;”的程序集“&lt;assemblyname&gt;”
构造函数间接引用包含 \<typename\> 的程序集“\<assemblyname\>”。 将对 \<filename\> 的文件引用添加到项目中。  
  
 表达式使用类、结构、接口、枚举或委托等类型，但程序集没有对定义该类型的程序集的项目引用。  
  
 **错误 ID：**BC31528  
  
### 更正此错误  
  
-   在项目属性中，添加对包含某程序集的文件的引用，该程序集定义正在使用的类型。
