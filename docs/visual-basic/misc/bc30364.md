# “Sub New”不能声明为“&lt;modifier&gt;”
已尝试使用修饰符声明 `Sub New`，例如 `Overrides`，但是无效。  
  
 **错误 ID：**BC30364  
  
### 更正此错误  
  
-   删除此修饰符。
