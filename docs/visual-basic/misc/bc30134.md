# <a name="labels-are-not-valid-in-the-immediate-window"></a>标签在即时窗口中无效
行标签可启用分支，但不允许用于调试上下文。  
  
 **错误 ID：** BC30134  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   不要在“即时” **** 窗口中输入行标签。
