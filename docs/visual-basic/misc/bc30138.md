# 未能在“&lt;filename&gt;”路径中创建临时文件：&lt;error message&gt;
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 编译器调用程序集链接器（Al.exe，也称作 Alink）生成包含清单的程序集。 在创建文件或对其写入内存资源时，该链接器报告了错误。 这可能是配置问题。  
  
 **错误 ID：**BC30138  
  
### 更正此错误  
  
1.  检查引用的错误消息并参考 [Al.exe 工具错误和警告](http://msdn.microsoft.com/zh-cn/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)主题，以获得进一步的解释和建议。  
  
2.  如果仍然出现错误，则收集有关该情况的信息并通知 Microsoft 产品支持服务。
