# 在常量表达式中不能出现前导“.”或“!”
成员访问 \(.\) 并且字典成员访问 \(\!\) 需要一个表达式，用于指定多数情况下包含该成员的元素，其中包括常量表达式。 以下声明无效。  
  
```  
' Not valid. Const c As String = .name  
```  
  
 **错误 ID：**BC30995  
  
### 更正此错误  
  
-   指定包含想要访问的成员的实例。
