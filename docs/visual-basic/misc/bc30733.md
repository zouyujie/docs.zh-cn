# <a name="39ltmembernamegt39-is-already-declared-by-39ltprocedurenamegt39-which-was-generated-for-this-ltcontainernamegt"></a>&lt;membername&gt;已通过声明&lt;过程名&gt;，其中为其生成此&lt;容器名称&gt;
所声明的某个成员与由公共语言运行时声明的某个成员同名。  
  
 **错误 ID：** BC30733  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改成员名称以消除冲突。
