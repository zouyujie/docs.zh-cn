# “GoTo &lt;labelname&gt;”无效，因为“&lt;labelname&gt;”位于不包含此语句的“SyncLock”语句内
你不能分支到 `SyncLock` 块。  
  
 **错误 ID：**BC30755  
  
### 更正此错误  
  
-   重构你的代码，以便标签位于 `SyncLock` 块之前。
