# 需要有以句点开头的标识符
匿名类型声明初始值设定项列表中包含无法推断出属性名称的值，没有为其分配属性名称。  
  
```  
' Not valid. ' Dim anon1 = New With {.grade = 100, 95}  
```  
  
 **错误 ID:** BC36576  
  
### 更正此错误  
  
-   为初始值设定项列表中的每个值提供属性名称，代码如下所示：  
  
    ```  
    Dim anon2 = New With {.grade1 = 100, .grade2 = 95}  
    ```
