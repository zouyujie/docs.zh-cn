# “Continue Do”只能出现在“Do”语句内
`Continue Do` 语句只能出现在 `Do...Loop` 循环内。  
  
 **错误 ID:** BC30782  
  
### 更正此错误  
  
1.  如果 `Continue Do` 语句在 `For...Next` 循环中，请将该语句更改为 `Continue For`。  
  
2.  如果 `Continue Do` 语句在 `While...End While` 循环中，请将该语句更改为 `Continue While`。  
  
3.  否则，请删除 `Continue Do` 语句。
