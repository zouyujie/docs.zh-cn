# 已声明 XML 命名空间前缀“prefix”
`Imports` 语句定义的 XML 命名空间前缀与之前定义的 XML 命名空间前缀匹配。  
  
 **错误 ID：**BC30573  
  
### 更正此错误  
  
-   对 XML 命名空间前缀使用唯一名称。
