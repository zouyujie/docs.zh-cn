# Imports 别名“&lt;name&gt;”与根命名空间中声明的“&lt;name2&gt;”冲突
`Imports` 语句中使用的别名名称与根命名空间中声明的另一个名称冲突。  
  
 **错误 ID：**BC31403  
  
### 更正此错误  
  
-   选择 `Imports` 语句使用的唯一别名名称。
