# “Exit For”只能出现在“For”语句内
`Exit For` 语句出现在 `For` 循环外部。`Exit For` 仅在 `For` 语句或 `For Each` 语句和相应 `Next` 语句之间有效。  
  
 **错误 ID：**BC30096  
  
### 更正此错误  
  
1.  确保 `Exit For` 之前有一个有效的 `For` 或 `For Each` 语句，之后有一个有效的 `Next` 语句。  
  
2.  验证 `For` 循环内的其他控制结构是否被正确终止。
