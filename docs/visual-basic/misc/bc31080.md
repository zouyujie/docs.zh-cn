# 没有为类型“&lt;typename1&gt;”和“&lt;typename2&gt;”定义运算符“&lt;operatorname&gt;”
没有为类型“\<typename1\>”和“\<typename2\>”定义运算符“\<operatorname\>”。 使用“Is”运算符比较两个引用类型。  
  
 试图以对指定类型不适合的方式使用运算符。 在使用“\=”运算符而不是 `Is` 运算符比较两个对象时，可能会导致此错误。  
  
 **错误 ID：**BC31080  
  
### 更正此错误  
  
1.  使用 `Is` 运算符比较两个引用类型。
