# 选项 &lt;optionname&gt; 后面只能跟“+”或“-”
该选项后的文本不是加号 \(\+\) 或减号 \(\-\)。  
  
 **错误 ID：**BC2009  
  
### 更正此错误  
  
-   将此选项后的文本替换为加号 \(\+\) 以打开该选项。  
  
     — 或 —  
  
-   将此选项后的文本替换为减号 \(\-\) 以关闭该选项。
