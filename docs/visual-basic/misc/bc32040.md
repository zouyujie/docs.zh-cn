# “&lt;keyword&gt;”关键字用于重载继承的成员；重载“Sub New”时不要使用“&lt;keyword&gt;”关键字
使用 `Overloads` 关键字声明了一个构造函数。  
  
 Visual Basic 不支持继承或重载构造函数。  
  
 **错误 ID：**BC32040  
  
### 更正此错误  
  
-   从所有构造函数声明中删除 `Overloads` 关键字。
