# &lt;typename&gt;“&lt;membername&gt;”必须声明为“Shadows”，因为另一个同名成员被声明为“Shadows”
此属性或过程的名称与使用 `Shadows` 修饰符标记的另一个属性或过程相同。  
  
 **错误 ID：**BC30695  
  
### 更正此错误  
  
-   使用 `Shadows` 修饰符。
