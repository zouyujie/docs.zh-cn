# 数组初始值设定项的维数太多。
在数组的初始值设定项中提供了过多维度。  
  
 **错误 ID：**BC30566  
  
### 更正此错误  
  
-   检查数组初始值设定项，以确定需要多少维数。  
  
-   删除多余的一个或多个参数。
