# “New”不能在声明为“MustInherit”的类上使用
不能直接实例化 `MustInherit` 类，因此不可在 `MustInherit` 类上使用 `New` 运算符。 虽然变量和值的编译时类型可以为 `MustInherit`，但此类变量和值必须为 null 值或包含对派生自 `MustInherit` 类型的常规类的实例的引用。  
  
 **错误 ID：**BC30569  
  
### 更正此错误  
  
-   删除 `New` 运算符。
