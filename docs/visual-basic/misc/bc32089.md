# “&lt;parametername&gt;”已被声明为此方法的类型参数
泛型过程使用与类型参数相同的名称定义普通参数或局部变量。  
  
 过程的每个参数（包括泛型过程的每个类型参数）必须具有与所有其他参数不同的名称。 由于过程参数被用作局部变量，因此在过程中声明的任何局部变量也必须具有与所有参数和类型参数不同的名称。  
  
 **错误 ID：**BC32089  
  
### 更正此错误  
  
-   更改普通参数或本地变量的名称。
