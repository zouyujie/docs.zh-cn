# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>注释语句不能评估 （不在目录）
在调试会话期间，已在即时窗口中输入注释语句。 未计算即时窗口中的注释语句。  
  
 **错误 ID：** BC30974
