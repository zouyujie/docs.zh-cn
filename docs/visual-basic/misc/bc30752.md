# “On Error”语句在“SyncLock”语句内无效
`On Error` 语句不可用于 `SyncLock` 块中，因为它们可能会中断线程同步。  
  
 **错误 ID：**BC30752  
  
### 更正此错误  
  
1.  将 `On Error` 置于 `SyncLock` 块外。
