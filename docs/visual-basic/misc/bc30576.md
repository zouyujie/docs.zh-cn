# 语句不声明“Get”或“Set”方法
你的语句无法提供围绕 `Property` 过程的 `Get` 或 `Set` 声明语句。 属性被定义为包含在 `Property` 和 `End Property` 语句中的代码块。 在此块中，每个 `Property` 过程显示为包含在声明语句和结束语句中的内部块。  
  
 **错误 ID：**BC30576  
  
### 更正此错误  
  
-   提供 `Get` 或 `Set` 声明语句。
