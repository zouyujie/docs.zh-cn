# “&lt;declaration1&gt;”无法替代“&lt;declaration2&gt;”，因为后者已声明为“NotOverridable”
过程或属性声明试图替代一个同名的继承元素，但该继承元素被指定为 `NotOverridable`。  
  
 **错误 ID：**BC30267  
  
### 更正此错误  
  
-   将 `NotOverridable` 关键字从继承元素的声明中删除，或删除替代声明。
