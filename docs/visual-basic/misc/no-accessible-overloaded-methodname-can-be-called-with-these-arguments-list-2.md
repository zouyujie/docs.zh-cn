# 必须进行收缩转换才能用这些参数调用可访问的重载“&lt;methodname&gt;”：&lt;list&gt;
调用了重载方法，但是该方法必须进行缩放转换才能与提供的参数列表匹配。  
  
### 更正此错误  
  
1.  指定 `Option``Strict` `Off`。  
  
2.  更改参数以匹配重载方法的签名。
