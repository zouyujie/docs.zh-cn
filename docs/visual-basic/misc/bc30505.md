# 实现接口成员的方法或事件不能声明为“Shared”
已尝试声明为实现接口成员的 `Shared` 方法或事件。 不能将类中实现的方法和事件指定为 `Shared` 或 `Private`，非可继承的类除外。  
  
 **错误 ID：**BC30505  
  
### 更正此错误  
  
-   删除 `Shared` 关键字。
