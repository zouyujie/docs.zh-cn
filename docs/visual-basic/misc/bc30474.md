# “&lt;method1&gt;”不能重写“&lt;method2&gt;”，因为它是“Declare”语句
你试图重写用 `Declare` 语句声明的基类名称上的委托。  
  
 **错误 ID：**BC30474  
  
### 更正此错误  
  
1.  更改重写的成员，使其不再是 `Declare` 语句。  
  
2.  请勿尝试重写此方法。
