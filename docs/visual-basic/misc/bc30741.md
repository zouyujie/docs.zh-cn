# “&lt;typename1&gt;”不能转换为类型“&lt;typename2&gt;”。
在两个不兼容的类型之间尝试执行了类型转换。  
  
 **错误 ID：**BC30741  
  
### 更正此错误  
  
-   验证正在被强制转换的类型。
