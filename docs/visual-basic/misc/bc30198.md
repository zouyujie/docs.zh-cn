# 应有“)”
列表（例如，过程定义或过程调用中的参数列表）不是用右括号 \(`)`\) 字符终止的。  
  
 **错误 ID：**BC30198  
  
### 更正此错误  
  
-   在列表末尾添加一个右括号字符。
