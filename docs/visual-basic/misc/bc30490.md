# “Default”不能与“&lt;specifier&gt;”组合
试图将 `Default` 关键字与此上下文中无效的说明符（例如 `Shared` 或 `Private`组合起来。  
  
 **错误 ID：**BC30490  
  
### 更正此错误  
  
-   删除 `Default`。  
  
     \- 或 \-  
  
-   删除冲突的说明符。
