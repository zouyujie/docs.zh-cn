# 带前缀的命名空间声明在 XML 文本中不能有空值
XML 文本中的 XML 命名空间声明不包括命名空间值。 例如，下面的代码将导致此错误：  
  
```vb#  
Dim book = <book xmlns:ns=""/>  
```  
  
 **错误 ID：**BC31184  
  
### 更正此错误  
  
-   在 XML 命名空间声明中包括有效的命名空间，或从 XML 文本中删除 XML 命名空间声明。  
  
     作为替代方法，你可以使用 `Imports` 语句以标识空命名空间的命名空间前缀。 例如:  
  
    ```vb#  
    Imports <xmlns:ns="">  
    ```
