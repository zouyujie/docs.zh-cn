# Sub 或 Set 中的“Return”语句不能返回值
`Sub` 过程和属性 `Set` 过程不能返回值。  
  
 **错误 ID：**BC30647  
  
### 更正此错误  
  
-   将当前的过程更改为一个函数，或者，如果当前的过程是一个属性的一部分，将其更改为一个 `Get` 属性过程。  
  
-   通过修改使用 `ByRef` 关键字的引用传递的参数值，你可以有效地从 `Sub` 返回值。
