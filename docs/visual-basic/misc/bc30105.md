# 索引数少于索引数组的维数
用于访问数组元素的索引数必须完全等同于数组的秩，即为其声明的维数。  
  
 **错误 ID：**BC30105  
  
### 更正此错误  
  
-   为数组引用添加下标，直到下标的总数等于数组的秩为止。
