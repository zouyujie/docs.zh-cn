# WithEvents 变量名称的长度不能超过 1019 个字符
使用 `WithEvents` 声明的每个变量的名称或标识符限制为 1019 个字符。  
  
 **错误 ID：**BC32205  
  
### 更正此错误  
  
-   减小标识符的长度。
