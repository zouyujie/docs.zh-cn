# 属性没有 Public 构造函数，因此不能使用。
正在使用的属性的构造函数是 `Private`，且无法使用。  
  
 **错误 ID：**BC30758  
  
### 更正此错误  
  
1.  如果看到你开发的自定义属性出现此错误，请将其 `Sub``New` 构造函数的访问修饰符更改为 `Public`。
