# <a name="39finally39-statements-are-not-valid-in-the-immediate-window"></a>最后语句中不能使用即时窗口
`Finally`语句仅可以在源代码中使用。  
  
 **错误 ID:** BC30716  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从调试代码中删除 `Finally` 语句。
