# “&lt;name1&gt;”与 &lt;type&gt; 冲突，因为“&lt;name2&gt;”中定义了相同名称
遇到与具有相同名称的类型冲突的名称。  
  
 **错误 ID：**BC31073  
  
### 更正此错误  
  
1.  声明唯一的名称。  
  
2.  在冲突成员名称两端使用方括号。
