# 从“System.&lt;classname&gt;”继承是无效的
类不能从 `System.Array`、`System.Delegate`、`System.Enum` 或 `System.ValueType` 继承。  
  
 **错误 ID：**BC30015  
  
### 更正此错误  
  
-   删除 `Inherits` 语句，将将其更改为从有效基类继承。
