# “End SyncLock”前面必须是匹配的“SyncLock”
`SyncLock` 块以 `SyncLock` 关键字开头，以 `End``SyncLock`构造结尾。  
  
 **错误 ID：**BC30674  
  
### 更正此错误  
  
-   确保 `SyncLock` 块以 `SyncLock` 语句开头。
