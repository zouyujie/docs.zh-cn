# <a name="39ltnamegt39-is-ambiguous-between-declarations-in-modules-39ltmodulename1gt39-and-39ltmodulename2gt39"></a>&lt;名称&gt;之间不明确的模块中的声明&lt;modulename1&gt;'和'&lt;modulename2&gt;
你提供的名称不明确，因此与另一个名称冲突。 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]编译器没有任何冲突解决规则; 您必须自己区分名。  
  
 **错误 ID：** BC30562  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   完全限定名称。
