# “End Module”前面必须是匹配的“Module”
Module 块以 `Module` 语句开头并以 `End Module` 构造结尾。  
  
 **错误 ID：**BC30622  
  
### 更正此错误  
  
-   请确保模块以 `Module` 语句开头。
