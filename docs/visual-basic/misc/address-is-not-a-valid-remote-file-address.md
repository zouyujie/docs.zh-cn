# &lt;address&gt; 不是有效的远程文件地址
\<address\> 不是有效的远程文件地址。 有效的地址应包括协议、路径和文件名。  
  
 为上载或下载操作指定了无效的地址。 地址必须包含协议（如 `ftp` 或 `http`）、有效路径和文件名。 此错误可能是错误分析文件路径的结果。  
  
### 更正此错误  
  
-   检查你指定的地址以确定出错的位置，并提供所需信息。
