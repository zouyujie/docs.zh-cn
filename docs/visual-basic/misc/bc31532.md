# 项目“&lt;项目名称1&gt;”对包含“&lt;类型名称&gt;”的项目“&lt;项目名称2&gt;”发起一个间接引用
项目“\<项目名称1\>”对包含“\<类型名称\>”的项目“\<项目名称2\>”发起一个间接引用。 将对“\<项目名称2\>”的项目引用添加到你的项目。  
  
 项目中的代码访问另一个项目中定义的类型，但你的项目不具有对该定义项目的直接引用。  
  
 类型可能是类、结构、接口、模块或枚举。  
  
 定义引用类型的项目生成包含该类型的程序集。 如果你的项目不直接引用该定义项目，编译器无法保证包含该类型的程序集是在解决方案中且可供访问。  
  
 **错误 ID：**BC31532  
  
### 更正此错误  
  
-   确定可定义引用类型的项目，并向其添加项目引用。
