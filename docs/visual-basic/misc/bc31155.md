# 带引号的特性值内不能出现表达式
带引号的特性值内不能出现表达式。 请尝试删除引号。  
  
 XML 文本的特性值中的嵌入式表达式包含在引号中。  
  
 **错误 ID：**BC31155  
  
### 更正此错误  
  
-   从特性值中删除分隔引号。 下面是一个示例：  
  
    ```vb#  
    ' Generates an error. Dim elem = <outer attr="<%= value %>" /> ' Does not generate an error. Dim elem = <outer attr=<%= value %> />  
    ```
