# 声明为“ReadOnly”的属性不能有“Set”
`Set` 过程写入属性的值。 不能写入 `ReadOnly` 属性。  
  
 **错误 ID：**BC30022  
  
### 更正此错误  
  
-   删除 `Property` 语句中的 `ReadOnly` 关键字，或者删除属性正文中的 `Set` 过程。
