# 重载决策失败，因为没有可使用这些参数调用的可访问“&lt;method&gt;”: &lt;error&gt;
你已调用重载方法，但编译器没有找到具有匹配参数列表的方法。  
  
 **错误 ID：**BC30518  
  
### 更正此错误  
  
-   检查所提供的参数列表。
