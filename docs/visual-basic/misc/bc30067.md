# “Exit Functio”在 Sub 或属性中无效
`Exit Function` 出现在 `Sub` 过程或 `Property` 过程中。`Exit` 语句必须与它所在的块匹配。  
  
 **错误 ID：**BC30067  
  
### 更正此错误  
  
-   根据需要，用 `Exit Sub` 或 `Exit Property` 语句替换 `Exit Function`。
