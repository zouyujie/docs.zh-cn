# “&lt;implementsclause&gt;”不能实现“&lt;typename&gt;”，因为“&lt;typename&gt;”是保留的名称
`Implements` 语句指定了保留的名称。  
  
 **错误 ID：**BC31415  
  
### 更正此错误  
  
-   重命名无法实现的接口成员。
