# 只能指定“Public”、“Private”、“Protected”、“Friend”或“Protected Friend”中的一个
在元素声明中发生冲突的可访问性。 只能指定一个可访问性。  
  
 **错误 ID：**BC30176  
  
### 更正此错误  
  
-   选择一个可访问性，并删除声明中的其他关键字。
