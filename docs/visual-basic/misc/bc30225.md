# 每个文件中只能出现一次“Option &lt;specifier&gt;”语句。
`Option Compare`、`Option Explicit` 和 `Option Strict` 语句只能在同一源文件中各使用一次。  
  
 **错误 ID：**BC30225  
  
### 更正此错误  
  
-   删除重复的 `Option` 语句。
