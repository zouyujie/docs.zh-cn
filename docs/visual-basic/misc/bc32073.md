# 模块不能为泛型。
`Module` 语句使用 `Of` 关键字来引入泛型类型参数。  
  
 你可以定义并使用泛型类、结构、接口、过程和委托。 不能定义泛型模块。  
  
 **错误 ID：**BC32073  
  
### 更正此错误  
  
1.  从 `Module` 语句中删除 `Of` 关键字。  
  
2.  如果想要使用泛型模块的功能，最接近的是泛型类。 使用 [Class 语句](../../visual-basic/language-reference/statements/class-statement.md) 而非 `Module` 语句。
