# 调用另一个构造函数时隐式引用尚未完成的对象是无效的
在对象构造函数完成构造对象之前，已引用对象成员。  
  
 **错误 ID：**BC31096  
  
### 更正此错误  
  
-   从一个构造函数调用另一个构造函数时，请勿使用 `MyBase`、`MyClass` 或 `Me`。
