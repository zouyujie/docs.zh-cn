# 形参“&lt;形参名称&gt;”已有匹配的实参
参数在同一过程调用中按名称传递了多次。  
  
 **错误 ID：**BC36567  
  
### 更正此错误  
  
-   检查以确定是否其中一个参数名拼写错误。  
  
-   从参数列表中删除重复的参数。
