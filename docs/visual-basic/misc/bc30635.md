# &quot;ReadOnly&quot; 不能与 &quot;WriteOnly&quot; 组合
`ReadOnly` 和 `WriteOnly` 修饰符互相排斥。  
  
 **错误 ID：**BC30635  
  
### 更正此错误  
  
-   指定 `ReadOnly` 或 `WriteOnly`，但不能同时指定这两者。
