# 从“&lt;typename1&gt;”到“&lt;typename2&gt;”的转换必须为显式转换
无法隐式实现此转换。  
  
 **错误 ID：**BC30664  
  
### 更正此错误  
  
-   使用 `CType` 或某个特定转换函数，如 `CInt` 或 `CDec`。
