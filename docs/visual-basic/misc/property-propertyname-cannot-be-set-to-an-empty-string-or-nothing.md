# 属性 &lt;propertyname&gt; 不能设置为空字符串或 Nothing
该属性要求使用除空字符串 \(""\) 或 `Nothing` 以外的值。 如果计算值时有误，则可能会出现此结果。  
  
### 更正此错误  
  
-   提供一个有效值。
