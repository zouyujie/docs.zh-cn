# 结构中的 &quot;Declare&quot; 语句不能声明为“&lt;specifier&gt;”
`Declare` 声明具有在 `Structure` 声明中无效的说明符。  
  
 **错误 ID：**BC30791  
  
### 更正此错误  
  
-   删除说明符。
