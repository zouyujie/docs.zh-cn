# 结构中包含“&lt;typename&gt;”类型的字段“&lt;fieldname&gt;”的文件 I/O 无效
试图向文件写入的结构包含不支持的类型。  
  
### 更正此错误  
  
-   检查结构来确定不支持的类型出现的位置，然后将其更改为有效类型。
