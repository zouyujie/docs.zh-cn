# 已在接口“&lt;interfacename&gt;”中声明方法“&lt;methodname&gt;”
你通过复制第二次声明了某个接口方法。  
  
 **错误 ID：**BC30377  
  
### 更正此错误  
  
-   删除重复的声明。
