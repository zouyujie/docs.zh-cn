# <a name="delegate-constructor-for-type-39lttypenamegt39-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>类型的委托构造函数&lt;typename&gt;'，找不到实现委托，有必要
包含该类型的程序集可能被损坏或无效。  
  
 **错误 ID：** BC31074  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  验证包含该类型的程序集是有效的。  
  
2.  再次编译此程序以查看错误是否重复出现。  
  
3.  如果仍出现错误，请保存你的工作并重新启动 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]。
