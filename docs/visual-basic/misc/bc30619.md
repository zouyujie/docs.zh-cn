# 语句不能出现在枚举体内
语句不能出现在枚举体内。 假定已到达枚举末尾。  
  
 遇到意外的语言构造。 假定缺少 `End Enum` 构造，而且此点后的所有源代码都在枚举外。  
  
 **错误 ID：**BC30619  
  
### 更正此错误  
  
1.  验证枚举中所用代码的语法是否正确。  
  
2.  确保接口定义以 `End Enum` 构造结束。
