# “With”必须以匹配的“End With”结束
出现 `With` 语句而没有相应的 `End With` 语句。 必须使用 `End With` 语句结束 `With` 块。  
  
 **错误 ID：**BC30085  
  
### 更正此错误  
  
-   如果此 `With` 块属于一组嵌套的 `With` 块，请确保每个块均已正确终止。  
  
-   将 `End With` 语句添加到 `With` 块末尾。
