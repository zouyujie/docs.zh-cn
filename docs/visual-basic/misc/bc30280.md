# 枚举“&lt;枚举名称&gt;”必须至少包含一个成员
发生空枚举。 在 `Enum` 语句和 `End Enum` 语句之间必须至少声明一个成员。  
  
 **错误 ID：**BC30280  
  
### 更正此错误  
  
1.  将成员声明添加到枚举。
