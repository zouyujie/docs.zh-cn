# 此类未实现接口“&lt;interfacename&gt;”
此类或结构的成员尝试实现类或结构未实现的接口的成员。  
  
 **错误 ID：**BC31035  
  
### 更正此错误  
  
-   在类或结构的开头添加 `Implements` 语句，使它实现相应的接口。  
  
-   从导致此错误的成员中删除 `Implements` 关键字。
