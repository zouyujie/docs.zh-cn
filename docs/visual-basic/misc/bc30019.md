# 委托无法处理事件
委托是一种引用类型，它指向共享过程或对象上的实例过程。 因为它所指向的过程可以通过赋值更改，所以 `Delegate` 语句不能支持 `Handles` 或 `Implements` 子句。  
  
 **错误 ID：**BC30019  
  
### 更正此错误  
  
-   从 `Delegate` 语句中删除 `Handles` 子句。
