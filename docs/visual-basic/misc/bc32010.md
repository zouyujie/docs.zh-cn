# “&lt;name&gt;”不是字段或属性，因此不能命名为特性说明符中的参数。
特性块设置该特性的非变量成员的值。 可以仅对变量成员（如字段或属性）赋值。  
  
 **错误 ID：**BC32010  
  
### 更正此错误  
  
1.  确保特性和成员的名称拼写正确。  
  
2.  如果该成员是非变量，则从特性块中删除参数。
