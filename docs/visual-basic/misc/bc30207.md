# “Option Compare”必须后接“Text”或“Binary”
`Option Compare` 语句包含错误设置或未含任何设置。`Option Compare` 中唯一允许使用的设置是 `Text` 和 `Binary`。  
  
 **错误 ID：**BC30207  
  
### 更正此错误  
  
1.  检查设置说明符是否存在拼写错误。  
  
2.  将 `Text` 或 `Binary` 添加到 `Option Compare` 语句；例如，`Option Compare Text`。
