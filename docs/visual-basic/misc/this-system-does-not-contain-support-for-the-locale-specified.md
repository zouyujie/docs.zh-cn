# 此系统不包含对指定区域设置的支持
你正尝试使用 `VbStrConv` 枚举，但你的系统不支持指定的区域设置。  
  
### 更正此错误  
  
1.  升级计算机以支持区域设置，或在支持该区域设置的计算机上运行该应用程序。
