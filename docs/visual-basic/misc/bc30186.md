# <a name="39using39-statements-are-not-valid-in-the-immediate-window"></a>'Using' 语句中不能使用即时窗口
`Using`语句仅可以在源代码中使用。  
  
 **错误 ID：** BC30186  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从“即时” `Using`**窗口中删除** 语句。
