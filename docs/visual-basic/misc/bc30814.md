# 不再支持“GoSub”语句
`GoSub` 不能用于调用过程。  
  
 **错误 ID：**BC30814  
  
### 更正此错误  
  
-   直接调用过程而不使用 `GoSub`；例如：  
  
    ```  
    CalculateInterest(Amount, Rate, Time)  
    ```
