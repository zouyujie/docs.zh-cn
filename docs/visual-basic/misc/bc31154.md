# XML 声明中不允许出现特性“attributeName”
为 XML 文档文本指定了无效的特性。 XML 文档文本的有效特性值基于 XML 1.0 规范。  
  
 **错误 ID：**BC31154  
  
### 更正此错误  
  
-   从 XML 文档文本中删除无效的特性。
