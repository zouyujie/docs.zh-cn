# 属性“&lt;propertyname&gt;”为“ReadOnly”
尝试为声明为 `ReadOnly` 的属性赋值。  
  
 **错误 ID：**BC30526  
  
### 更正此错误  
  
-   从属性声明中删除 `ReadOnly` 说明符。
