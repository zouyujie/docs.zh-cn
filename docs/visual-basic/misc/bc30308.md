# “&lt;method1&gt;”无法重写“&lt;method2&gt;”，因为它们在可选参数上存在差异
已尝试用另一种方法重写第一种方法，另一种方法与第一种方法在其可选参数值上存在差异，也就是说，它们的签名不同。 类型可重写继承的可重写方法，方法是声明具有相同名称和签名的方法，并用 `Overrides` 修饰符标记声明。  
  
 **错误 ID：**BC30308  
  
### 更正此错误  
  
-   请确保这两种方法具有相同的签名。
