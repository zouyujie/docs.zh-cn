# 模块不能声明为“&lt;specifier&gt;”
已使用无效的访问修饰符声明模块。  
  
 **错误 ID:**  BC31052  
  
### 更正此错误  
  
-   用有效的访问修饰符（如 `Public` 或 `Friend`）替换该访问修饰符。
