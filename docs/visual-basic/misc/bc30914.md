# “&lt;name&gt;”无法公开正在通过 &lt;specifier2&gt;“&lt;type2&gt;”向&lt;specifier1&gt;“&lt;type1&gt;”实现的事件的基础委托类型“&lt;delegatetype&gt;”
该代码在公共类外部公开了一个私有类型。  
  
 **错误 ID：**BC30914  
  
### 更正此错误  
  
-   声明类型 `Public` 或使用另一种类型。
