# “Next”前面必须是匹配的“For”
出现 `Next` 语句而没有相应的 `For` 或 `For Each` 语句。`Next` 前面必须有相应的 `For` 或 `For Each` 语句。  
  
 **错误 ID：**BC30092  
  
### 更正此错误  
  
1.  如果此 `For` 循环是一组嵌套 `For` 循环的一部分，请确保每个循环正常终止。  
  
2.  验证 `For` 循环内的其他控制结构是否被正确终止。  
  
3.  确保此 `For` 循环的格式正确。
