# “End While”前面必须是匹配的“While”
出现 `End While` 语句而没有相应的 `While` 语句。`End While` 前面必须有相应的 `While` 语句。  
  
 **错误 ID：**BC30090  
  
### 更正此错误  
  
1.  如果此 `While` 块属于一组嵌套的 `While` 块，请确保每个块均已正确终止。  
  
2.  验证 `While` 块中的其他控制结构是否被正确终止。  
  
3.  确保此 `While` 块的格式正确。
