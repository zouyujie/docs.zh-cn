# 不能使用 &lt;number&gt; 参数调用方法“&lt;methodname&gt;”
方法调用包含错误的参数数量。  
  
### 更正此错误  
  
1.  对你已提供的参数进行计数。  
  
2.  将数量与你想调用的方法中的参数的数量进行匹配。  
  
3.  如果两个数量不一致，请调整参数的数量。
