# 类型“&lt;type1&gt;”的值无法转换为“&lt;type2&gt;”，原因是数组类型的维数不同
试图将一个数组转换为另一个具有不同维数的数组。  
  
 **错误 ID：**BC30414  
  
### 更正此错误  
  
-   更改一个数组或同时更改两个数组，使两者的维数匹配。
