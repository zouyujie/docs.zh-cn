# XML 注释参数“&lt;parameter&gt;”与相应的 &lt;keyword&gt; 语句上的参数不匹配
XML 注释参数 \<parameter\> 与相应的 \<keyword\> 语句上的参数不匹配。 将忽略 XML 注释。  
  
 为方法中的某些参数（而不是全部）指定了 XML 注释标记。  
  
 **错误 ID：**BC42307  
  
### 更正此错误  
  
-   为方法中的所有参数提供 XML 注释标记。
