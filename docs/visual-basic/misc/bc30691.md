# “&lt;typename&gt;” 是“&lt;containername&gt;”中的一个类型，不能用作表达式
使用中的其中一个名称已声明为类型的名称，且不能在表达式中使用。  
  
 **错误 ID：**BC30691  
  
### 更正此错误  
  
-   对成员进行唯一命名。  
  
-   用方括号括起成员名称。
