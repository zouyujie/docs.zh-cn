# 应有“.”
代码具有包含感叹号 \(`!`\) 的 `Handles` 子句。  
  
 **错误 ID：**BC30287  
  
### 更正此错误  
  
1.  如果 `Handles` 子句引用对象内的事件，请使用句点 \(`.`\) 将对象与该事件分隔开。
