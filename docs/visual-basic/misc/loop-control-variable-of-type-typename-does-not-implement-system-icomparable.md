# 类型“&lt;typename&gt;”的循环控制变量不实现“System.IComparable”接口
变量为不实现 `System.IComparable` 接口的类型。`System.IComparable` 必须由值类型或类实现。  
  
### 更正此错误  
  
1.  请确保变量是值类型。
