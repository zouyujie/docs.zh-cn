# “End Structure”前面必须是匹配的“Structure”
`End Structure` 语句只能在以 `Structure` 语句开头的结构块末尾使用。  
  
 **错误 ID：**BC30621  
  
### 更正此错误  
  
-   确保当前结构块以 `Structure` 语句开头。
