# 在项目“&lt;projectname2&gt;”未引用的项目“&lt;projectname1&gt;”中声明了“&lt;name&gt;”
在当前项目未引用的项目中定义了一个编程元素。  
  
 **错误 ID：**BC32004  
  
### 更正此错误  
  
-   在当前项目中，添加对包含所需元素的项目的引用。
