# “Delegate”后面应为“Sub”或“Function”
`Delegate` 语句未指定 `Sub` 或 `Function` 过程。`Sub` 或 `Function` 关键字必须紧跟 `Delegate` 关键字。  
  
 **错误 ID：**BC30278  
  
### 更正此错误  
  
1.  在 `Delegate` 后紧跟 `Sub` 或 `Function` 关键字。  
  
2.  根据需要，指定过程名称、参数列表和返回类型。
