# 数组初始值设定项仅对数组有效，但“&lt;variablename&gt;”的类型是“&lt;typename&gt;”
尝试用值列表初始化非数组变量。  
  
 **错误 ID：**BC30679  
  
### 更正此错误  
  
-   将变量声明并初始化为数组；例如：  
  
     `Dim intarray As Integer() = {1, 5, 9}`  
  
-   将变量初始化为一个值；例如：  
  
     `Dim intvalue As Integer = 1`
