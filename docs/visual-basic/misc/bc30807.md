# 不再支持“Let”和“Set”赋值语句
不能在赋值中使用 `Let` 和 `Set`。  
  
 **错误 ID：**BC30807  
  
### 更正此错误  
  
-   省略赋值语句中的 `Let` 和 `Set`；例如：  
  
     `X = 2`
