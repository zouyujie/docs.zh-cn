# 参数“&lt;argumentname1&gt;”必须小于或等于参数“&lt;argumentname2&gt;”的长度
提供的参数的长度超过它要替换的参数的长度。  
  
### 更正此错误  
  
-   使第一个参数的长度小于或等于第二个参数的长度。
