# 数组初始值设定项的维数太少
在数组的初始值设定项中没有提供足够的维数。  
  
 **错误 ID：**BC30565  
  
### 更正此错误  
  
-   检查数组初始值设定项，以确定需要多少维数。  
  
-   提供缺少的一个或多个参数。
