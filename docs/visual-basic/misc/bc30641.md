# &quot;ByVal&quot; 不能与 &quot;ByRef&quot; 组合
`ByVal` 和 `ByRef` 关键字互斥。  
  
 **错误 ID：**BC30641  
  
### 更正此错误  
  
-   指定 `ByVal` 或 `ByRef`，但不能同时指定这两者。
