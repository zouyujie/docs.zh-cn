# “End With”前面必须是匹配的“With”
出现 `End With` 语句而没有相应的 `With` 语句。`End With` 前面必须有相应的 `With` 语句。  
  
 **错误 ID：**BC30093  
  
### 更正此错误  
  
1.  如果此 `With` 块属于一组嵌套的 `With` 块，请确保每个块均已正确终止。  
  
2.  验证 `With` 块中的其他控制结构是否被正确终止。  
  
3.  确保此 `With` 块的格式正确。
