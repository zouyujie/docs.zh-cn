# 应为“Dim”、“Const”、“Public”、“Private”、“Protected”、“Friend”、“Shadows”、“ReadOnly”或“Shared”中的一个
声明语句中缺少声明关键字。 一个可能的原因是，特性声明调用了方法。  
  
 **错误 ID：**BC30195  
  
### 更正此错误  
  
1.  检查是否在特性声明的内部声明了方法。  
  
2.  在该语句开始处使用适当的声明关键字。
