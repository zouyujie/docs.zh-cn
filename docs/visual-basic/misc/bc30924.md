# “&lt;name&gt;”不能公开事件的委托类型“&lt;delegatetype&gt;”中所用的类型“&lt;type1&gt;”，它正通过 &lt;specifier&gt; “&lt;type2&gt;”在项目外部实现
该代码在公共类外部公开了一个私有类型。  
  
 **错误 ID:** BC30924  
  
### 更正此错误  
  
-   声明类型 `Public` 或使用另一种类型。
