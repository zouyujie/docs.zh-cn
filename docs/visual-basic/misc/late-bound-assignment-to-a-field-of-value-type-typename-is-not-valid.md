# 当“&lt;name&gt;”是后期绑定表达式的结果时，对值类型为“&lt;typename&gt;”的字段的后期绑定赋值无效
试图进行的后期绑定赋值无效。  
  
### 更正此错误  
  
-   进行早期绑定赋值。
