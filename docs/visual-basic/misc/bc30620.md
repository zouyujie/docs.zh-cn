# “Option Strict”的后面只能跟“On”或“Off”
`Option Strict` 语句必须指定 `On` 或 `Off`。 其他所有值均无效。  
  
 **错误 ID：**BC30620  
  
### 更正此错误  
  
-   为 `Option Strict` 语句指定 `On` 或 `Off`。
