# “&lt;typename&gt;”不是类，因此不能用作特性
尝试将不是类的类型用作特性。  
  
 **错误 ID：**BC31503  
  
### 更正此错误  
  
1.  将自定义特性定义为从 `System.Attribute` 派生的类。
