# “&lt;qualifiedmethod&gt;”是“&lt;modifier&gt;”，因此它在此上下文中不可访问
尝试访问一个由于应用了修饰符而在此上下文中不可访问的方法。  
  
 **错误 ID：**BC30390  
  
### 更正此错误  
  
-   删除此修饰符。
