# “End Class”前面必须是匹配的“Class”
`End Class` 用于完成 `Class` 块，因此它只能出现在块的末尾。 有冗余的 `End Class`，或者 `End Class` 语句出现在其对应 `Class` 块的边界之外。  
  
 **错误 ID：**BC30460  
  
### 更正此错误  
  
-   找到并删除任何多余的 `End Class` 语句。  
  
-   将 `End Class` 语句移到代码中的适当位置。
