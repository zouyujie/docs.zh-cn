# 资源文件中的格式无效
你的项目中的资源文件不符合 Windows 资源文件标准格式或资源文件和 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 项之间存在不匹配。  
  
### 更正此错误  
  
1.  重新创建资源源和资源文件，并遵循 Windows 软件开发工具包中所述的语法。  
  
2.  从项目中删除无效的资源文件。
