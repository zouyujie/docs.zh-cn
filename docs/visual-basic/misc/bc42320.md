# 无法包括文件“&lt;commentfile&gt;”的 XML 段落“&lt;elementpath&gt;”
XML 注释中的 `<include>` 标记包含不存在元素的 XPath 表达式，或此文件包含格式不正确的 XML。  
  
 **错误 ID：**BC42320  
  
### 更正此错误  
  
1.  验证该文件中的 XML 是否有效。  
  
2.  验证文件中是否存在 XML 片段。
