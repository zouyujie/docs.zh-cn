# “End Event”前面必须是匹配的“Custom Event”
出现 `End Event` 语句而没有相应的 `Custom Event` 语句。`End Event` 前面必须有相应的 `Custom Event` 语句。  
  
 **错误 ID：**BC31123  
  
### 更正此错误  
  
-   确保前面的 `Custom Event` 语句有效且拼写正确。
