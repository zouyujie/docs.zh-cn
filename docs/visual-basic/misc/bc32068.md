# 泛型方法中的局部变量不能声明为“Static”
泛型过程中的局部变量的声明指定 `Static`。  
  
 Visual Basic 和 .NET Framework 当前不支持静态变量与泛型过程的任何组合。  
  
 **错误 ID：**BC32068  
  
### 更正此错误  
  
-   从变量声明中删除 `Static` 关键字。
