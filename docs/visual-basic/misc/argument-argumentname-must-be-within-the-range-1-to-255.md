# 参数“&lt;argumentname&gt;”必须在 1 到 255 的范围内
参数无效，因为它超出了 0 到 255 的范围。  
  
### 更正此错误  
  
1.  检查表达式中参数的拼写是否正确。 拼写错误的变量名可能会隐式创建一个初始化为零的数值变量。  
  
2.  检查之前对表达式中的变量进行的操作，尤其是那些从其他过程作为参数传递给该过程的操作。
