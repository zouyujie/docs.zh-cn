# 属性“&lt;propertyname&gt;”不能声明为“默认”，因为它会覆盖基 &lt;type&gt;“&lt;typename&gt;”上非默认的属性
已尝试声明属性 `Default`，而此声明无效，因为它会覆盖基类上尚未声明为 `Default` 的属性。  
  
 **错误 ID：**BC30504  
  
### 更正此错误  
  
-   删除 `Default` 声明。
