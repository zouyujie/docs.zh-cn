# 变量“&lt;variablename&gt;”在赋值前已被使用（Visual Basic 错误）
变量在赋值前已被使用，导致 <xref:System.NullReferenceException>。  
  
 **错误 ID：**BC42109  
  
### 更正此错误  
  
-   请确保已为变量赋值。
