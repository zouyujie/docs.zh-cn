# 模块中的成员不能声明为“&lt;specifier&gt;”
你使用了在 `Module` 语句内的成员上无效的说明符。 模块永远不能实例化，不支持继承且不能实现接口。  
  
 **错误 ID：**BC30436  
  
### 更正此错误  
  
-   删除说明符。
