# “&lt;typename&gt;”已声明为“MustInherit”，因此不能用作特性
自定义特性类不能声明为 `MustInherit`。  
  
 **错误 ID：**BC31506  
  
### 更正此错误  
  
-   从自定义特性中删除 `MustInherit` 修饰符。
