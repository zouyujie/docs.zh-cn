# “Imports”语句必须位于任何声明之前
`Imports` 语句位于源文件中的声明语句之后。  
  
 `Imports` 语句从被引用项目和程序集导入命名空间名称，以及在它所出现的项目中定义的命名空间名称。`Imports` 语句必须置于源代码文件中的任何标识符引用之前。  
  
 **错误 ID：**BC30465  
  
### 更正此错误  
  
-   将 `Imports` 语句移动到源文件顶部任何声明语句之前。
