# 说明符仅在声明的开头有效
指定可访问性和其他指定的关键字必须出现在声明关键字之前，例如 `Dim` 或 `Function`。  
  
 **错误 ID：**BC30181  
  
### 更正此错误  
  
-   移动特性列表（如果有）和声明关键字之间的所有说明符。
