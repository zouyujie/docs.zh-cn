# “&lt;variablename&gt;”循环控制变量已被封闭“For”循环使用
所有嵌套 `For` 循环必须使用唯一的控制变量。  
  
 **错误 ID：**BC30069  
  
### 更正此错误  
  
-   更改其中一个控制变量名。
