# XML 命名空间前缀“prefix”保留供 XML 使用，并且无法更改命名空间 URI
一个保留的 XML 命名空间前缀（如 `xmlns`）已用于 XML 文本中。  
  
 **错误 ID：**BC31152  
  
### 更正此错误  
  
-   将 XML 命名空间前缀更改为非保留的值。
