# “Catch”必须以匹配的“End Try”结束
代码中出现的 `Catch` 语句出现没有匹配的 `End Try` 语句。`Catch` 语句必须以 `End Try` 语句结束。  
  
 **错误 ID：**BC30441  
  
### 更正此错误  
  
1.  删除 `Catch` 语句。  
  
2.  添加 `End Try` 语句来结束块。
