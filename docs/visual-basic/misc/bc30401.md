# “&lt;interfacename1&gt;”无法实现“&lt;methodname&gt;”，因为在接口“&lt;interfacename2&gt;”上没有匹配的 &lt;method&gt;
当第一个接口继承的接口上没有匹配的方法时，你已尝试在该接口上实现方法。  
  
 **错误 ID：**BC30401  
  
### 更正此错误  
  
-   请确保你没有键错方法的名称或提供缺少的方法。
