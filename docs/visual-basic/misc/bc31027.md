# 无法打开文件“&lt;filename&gt;”: &lt;error&gt;
在此项目中打开文件时出错。  
  
 **错误 ID：**BC31027  
  
### 更正此错误  
  
1.  验证该文件是否存在，并且未损坏或是由另一个应用程序锁定。
