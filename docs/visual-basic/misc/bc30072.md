# “Case”只能出现在“Select Case”语句内
`Case` 语句出现在 `Select` 块之外。`Case` 语句只能用在 `Select` 语句或 `Select Case` 语句和其对应的 `End Select` 语句之间。  
  
 **错误 ID：**BC30072  
  
### 更正此错误  
  
-   删除 `Case` 语句或将其移动到 `Select` 块之内。
