# 非法的调用表达式或索引表达式
括号中的值遵循不是过程、属性或数组的表达式。  
  
 以下代码会生成此错误。  
  
 `Dim testVariable As Object = Nothing(1)`  
  
 因为 `Nothing` 不采用参数或索引，所以不能将其用于括号。  
  
 **错误 ID：**BC32303  
  
### 更正此错误  
  
-   删除圆括号及其包含的任何值。
