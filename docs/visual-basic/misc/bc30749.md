# <a name="unable-to-access-member"></a>不能访问成员
无法访问指定的文件。  
  
 **错误 ID：** BC30749  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  验证是否定义了成员并且已正确拼写了成员名称。  
  
2.  尝试访问该模块中声明的其中一个成员。 在某些情况下，调试环境找不到成员，因为尚未加载在其中声明成员的模块。
