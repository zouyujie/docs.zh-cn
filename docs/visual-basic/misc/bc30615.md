# 在类库项目中不能使用“End”语句
用于创建 Dll 的类库项目不允许使用 `End` 关键字来停止执行过程中的代码。  
  
 **错误 ID:** BC30615  
  
### 更正此错误  
  
-   使用控制结构，如 `While` 和 `For` 来控制程序执行流程。
