# &lt;namespace&gt; “&lt;namespacename&gt;”中的 &lt;type1&gt;“&lt;typename&gt;”和 &lt;type2&gt;“&lt;typename&gt;”冲突
已声明的两个元素（如类和模块）在同一命名空间中具有相同的名称。  
  
 **错误 ID：**BC30179  
  
### 更正此错误  
  
-   将其中一个元素的声明移到其他命名空间。  
  
-   更改其中一个元素的名称。
