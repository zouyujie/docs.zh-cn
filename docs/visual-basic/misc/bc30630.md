# “End Get”前面必须是匹配的“Get”
`End Get` 用于终止 `Get` 属性过程。`End Get` 构造出现在 `Get` 属性过程外。  
  
 **错误 ID：**BC30630  
  
### 更正此错误  
  
1.  请确保在 `Property` 关键字之后，`End Property` 构造之前声明 `Get` 属性过程。  
  
2.  请确保 `Get` 属性过程以 `Get` 关键字开始，以 `End Get` 构造结束。
