# 模块中的类型不能声明为“&lt;specifier&gt;”
在模块中声明了具有无效修饰符的类型。  
  
 **错误 ID：**BC30735  
  
### 更正此错误  
  
-   不要对模块中声明的类型使用 `Protected` 或 `Protected Friend` 修饰符。
