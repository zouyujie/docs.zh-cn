# 结构方法内部的局部变量不能声明为 &quot;Static&quot;
不能将 `Static` 修饰符用于结构中的局部变量。  
  
 **错误 ID：**BC31400  
  
### 更正此错误  
  
1.  从局部变量中删除 `Static` 修饰符。  
  
2.  将该变量声明为具有扩大范围的静态变量。
