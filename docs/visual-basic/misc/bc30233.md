# “&lt;specifier&gt;”在常量声明中无效
`Const` 语句包含无效的关键字。`Const` 语句只能包含 `Friend`、`Private`、`Protected`、`Public` 或 `Shadows` 关键字。  
  
 **错误 ID：**BC30233  
  
### 更正此错误  
  
1.  确保关键字拼写正确。  
  
2.  从 `Const` 语句中删除无效的关键字。
