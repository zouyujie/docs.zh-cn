# <a name="conversion-operators-cannot-convert-to-an-interface-type"></a>已使用接口类型将转换运算符声明为返回类型
通过接口类型将转换运算符声明为返回类型。  
  
 在编译时[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]认为到任何接口中存在任何引用类型从预定义的转换。 这种转换在运行时可能会失败，但编译器无法预知运行时结果，因此它允许编译任何这种转换。  
  
 由于编译器认为已定义此转换，因此它不允许你对其重新定义。  
  
 **错误 ID：** BC33025  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   完全删除此运算符定义。 已对其进行预定义。
