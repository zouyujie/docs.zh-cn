# 没有为“&lt;typename&gt;”中定义的扩展方法“&lt;methodname&gt;”的形参“&lt;parametername&gt;”指定实参
尚未为扩展方法中所需的形参提供实参。  
  
 **错误 ID：**BC36586  
  
### 更正此错误  
  
-   为指定形参提供实参。
