# “ReadOnly”变量不能作为赋值目标
在向其赋值的上下文中找到了 `ReadOnly` 属性。 执行过程中只能向可写的变量、属性和数组元素赋值。  
  
 **错误 ID：**BC30064  
  
### 更正此错误  
  
-   从声明变量的 `Dim` 语句删除 `ReadOnly` 关键字，或者删除对其赋值的语句。
