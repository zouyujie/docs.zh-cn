# 没有方法“&lt;methodname&gt;”可以接受参数“&lt;parametername&gt;”的类型“&lt;typename&gt;”的参数。
方法调用包含无效的类型的参数。  
  
### 更正此错误  
  
1.  检查参数的类型。  
  
2.  针对方法的实参类型匹配形参类型，以确保它们一致。 必要时更改类型。
