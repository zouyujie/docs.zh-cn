# 值“&lt;值名称1&gt;”不能转换为“&lt;值名称2&gt;”
在两个不兼容的类型之间尝试执行了类型转换。  
  
 **错误 ID：**BC30742  
  
### 更正此错误  
  
-   验证正在被强制转换的类型。
