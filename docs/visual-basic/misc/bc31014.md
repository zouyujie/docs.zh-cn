# 未能在 DLL“&lt;filename&gt;”中找到入口点“&lt;name&gt;”：&lt;error&gt;
未能在指定的 DLL 中找到过程名称。  
  
 **错误 ID：**BC31014  
  
### 更正此错误  
  
1.  验证过程名称是否拼写正确，以及大小写是否与 DLL 中函数的大小写匹配。
