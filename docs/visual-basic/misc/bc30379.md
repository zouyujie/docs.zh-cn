# 在 &quot;Try&quot; 语句中，&quot;Catch&quot; 不能出现在 &quot;Finally&quot; 之后
`Catch` 语句出现在代码中结束 `Try` 语句块的 `Finally` 之后。`Catch` 必须出现在 `Try...Catch...Finally` 语句块之内。  
  
 **错误 ID：**BC30379  
  
### 更正此错误  
  
1.  将 `Catch` 语句移到代码中更合适的位置。
