# “&lt;name&gt;”不能通过 &lt;name&gt; 在 &lt;specifier&gt; &lt;type2&gt;“&lt;typename&gt;”中公开类型 &lt;type1&gt;
试图公开公共类 `MyClass` 外部的私有类型。  
  
 **错误 ID：**BC30508  
  
### 更正此错误  
  
-   将该类型声明为 `Public`。
