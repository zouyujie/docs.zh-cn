# “Loop”和匹配的“Do”不能同时具有条件
`Loop` 语句包含 `While` 或 `Until` 子句，相应的 `Do` 语句也包含此类子句。 一个循环中只有 `Do` 和 `Loop` 中的其中一个语句可以指定条件。  
  
 **错误 ID：**BC30238  
  
### 更正此错误  
  
-   从 `Do` 语句或 `Loop` 语句中删除 `While` 或 `Until` 子句。
