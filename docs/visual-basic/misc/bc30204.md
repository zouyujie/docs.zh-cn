# 应为整数常量
出现 `#ExternalSource` 指令，其中第二个参数不是整数文本。 在此上下文中，仅整数文本有效。 已命名的常量或枚举成员无效。  
  
 **错误 ID：**BC30204  
  
### 更正此错误  
  
1.  使用已命名的常量或枚举成员，而非文本。  
  
2.  提供一个整数文本作为 `#ExternalSource` 指令的第二个参数。
