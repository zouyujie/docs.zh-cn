# 为“&lt;eventname&gt;”隐式定义的“&lt;member&gt;”不能隐藏基 &lt;class&gt;“&lt;classname&gt;”中的 &quot;MustOverride&quot; 方法
指定的事件隐式声明了一个成员，而此成员与使用 `MustOverride` 修饰符声明的方法同名。  
  
 **错误 ID：**BC31413  
  
### 更正此错误  
  
-   从基类的方法中删除 `MustOverride` 修饰符，或为属性或方法提供唯一名称。
