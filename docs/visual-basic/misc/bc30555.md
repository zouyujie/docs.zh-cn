# 默认成员“&lt;membername&gt;”不是属性
你尝试将成员声明为 `Default`。 但是，`Default` 关键字是用于标识默认属性的。  
  
 **错误 ID：**BC30555  
  
### 更正此错误  
  
-   检查以确保用 `Default` 标志的是属性，或删除 `Default` 关键字。
