# “Exit Sub”在函数或属性中无效
`Exit Sub` 出现在 `Function` 过程或 `Property` 过程中。`Exit` 语句必须与它所在的块匹配。  
  
 **错误 ID：**BC30065  
  
### 更正此错误  
  
-   根据需要，用 `Exit Function` 或 `Exit Property` 语句替换 `Exit Sub`。
