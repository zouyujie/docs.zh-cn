# “AddressOf”表达式在“Select Case”语句的第一个表达式中无效
不能对 `Select Case` 语句中的测试表达式使用 `AddressOf` 表达式。`AddressOf` 表达式返回函数委托，而 `Select Case` 语句的测试表达式必须是基本数据类型。  
  
 **错误 ID：**BC36636  
  
### 更正此错误  
  
-   检查你的代码以确定是否可以使用其他条件构造，例如 `If...Then...Else` 语句。
