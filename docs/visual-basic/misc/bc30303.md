# “Sub”不返回值，因此在“Sub”声明中不能使用类型字符
`Sub` 过程（如 `Function` 过程）是可以采用参数并执行一系列语句的单独过程。 与 `Function` 过程不同，`Sub` 不返回值，因此不能包含类型声明。  
  
 **错误 ID：**BC30303  
  
### 更正此错误  
  
-   将 `Function` 过程改为 `Sub` 过程。
