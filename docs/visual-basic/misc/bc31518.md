# 程序集“&lt;assemblyname&gt;”的路径长度超过 259 个字符，因此无法创建该程序集
此程序集的路径长度超过了最大长度 259 个字符。  
  
 **错误 ID：**BC31518  
  
### 更正此错误  
  
-   通过使用较短名称或选择较平坦的文件层次结构创建较短的路径。
