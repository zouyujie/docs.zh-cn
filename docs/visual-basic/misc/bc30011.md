# 内部编译器错误: 代码生成器收到格式不正确的输入
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 编译器无法成功解释它的某个内部编写文件。  
  
 **错误 ID：**BC30011  
  
### 更正此错误  
  
1.  再次编译此程序以查看错误是否重复出现。  
  
2.  如果错误重复出现，请重新安装 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 编译器。  
  
3.  如果重新安装后仍然出现错误，请收集有关该情况的信息并通知 Microsoft 产品支持服务。
