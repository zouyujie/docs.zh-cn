# “ReadOnly”属性必须提供“Get”
如果某个属性声明为 `ReadOnly`，则它必须提供用于读取其值的过程。  
  
 **错误 ID：**BC30126  
  
### 更正此错误  
  
1.  请确保 `Property` 语句和 `End Property` 语句之间包含 `Get` 过程。  
  
2.  验证 `Property` 声明中的其他过程是否正确终止。
