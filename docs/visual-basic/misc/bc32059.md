# 数组的下限只能是“0”
声明语句或 `New` 子句用于指定下限为非零的数组。  
  
 当创建或初始化数组变量时，你可以选择指定每个维度的上限。 如果执行此操作，则该维度的长度就成为上限加 1，因为下限始终为零。 你也可以选择指定下限，但必须指定 0。 这样做的优点是你的代码易于阅读。  
  
 **错误 ID：**BC32059  
  
### 更正此错误  
  
-   将下限指定更改为 0，或将其完全删除。
