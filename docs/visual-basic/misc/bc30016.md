# 标签在方法/多行 lambda 外无效
仅可在 `Sub`、`Function`、属性 `Get`、或属性 `Set` 过程中向语句添加标签。 只有可执行语句可以有标签，并且所有可执行语句必须处于过程内。  
  
 **错误 ID：**BC30016  
  
### 更正此错误  
  
1.  从该语句删除的标签或将语句移动到过程内。
