# 自动化操作期间未找到文件名或类名 (Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

未能找到在对 `GetObject` 函数的调用中为文件名或类指定的名称。  
  
### 更正此错误  
  
-   检查名称并重试。  确保用于 `class` 参数的名称与向系统注册的名称相匹配。
