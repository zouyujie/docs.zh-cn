# 行太长
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

源文本行不能超过 65535 个字符。  
  
 **错误 ID：**BC30494  
  
### 更正此错误  
  
-   将行的长度缩短到 65535 个字符或更少。
