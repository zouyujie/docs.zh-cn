# 找不到属性
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

该对象不支持所指定的属性。  
  
### 更正此错误  
  
1.  检查属性名的拼写。  
  
2.  检查对象的文档，确保当对象实际支持“caption”或类似的命名属性时您没有尝试访问类似“text”属性这样的内容。
