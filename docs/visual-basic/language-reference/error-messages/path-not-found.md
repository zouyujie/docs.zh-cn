# 找不到路径
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

在文件访问或磁盘访问操作中，操作系统无法找到指定路径。  文件的路径包括驱动器说明加上必须遍历以定位文件的目录和子目录。  路径可以是相对路径或绝对路径。  
  
### 更正此错误  
  
-   验证并重新指定路径。
