# 下标超出范围 (Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

数组下标无效，因为它不在允许的范围内。  维度的最小下标值始终是 0，最大下标值是由该维的 `GetUpperBound` 方法返回的。  
  
### 更正此错误  
  
-   更改下标，使其在有效的范围内。
