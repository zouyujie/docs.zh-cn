# 此关键字已与该集合的某个元素相关联
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

为某个集合成员指定的关键字已经标识集合的另一成员。  关键字是在 `Add` 方法中指定的字符串，用于唯一标识集合的特定成员。  
  
### 更正此错误  
  
-   对该成员使用其他关键字。
