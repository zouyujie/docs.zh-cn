# 溢出（Visual Basic 错误）
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

文本表示的值的大小超出了赋予该文本的数据类型的限制。  
  
 **错误 ID：**BC30036  
  
### 更正此错误  
  
-   查询目标数据类型的取值范围，然后重写该文本，使其符合此范围。
