# XML 轴属性不支持后期绑定
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

针对非类型化对象引用了 XML 轴属性。  
  
 **错误 ID：**BC31168  
  
### 更正此错误  
  
-   在引用 XML 轴属性之前，确保对象是强类型的 <xref:System.Xml.Linq.XElement> 对象。
