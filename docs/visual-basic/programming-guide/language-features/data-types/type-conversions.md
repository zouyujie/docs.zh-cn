# <a name="type-conversions-in-visual-basic"></a>Visual Basic 中的类型转换
将值从一种数据类型更改为另一种类型的过程被称为*转换*。 转换都是*扩大*或*收缩*，取决于所涉及的类型的数据容量。 它们也是*隐式*或*显式*，取决于在源代码中的语法。
