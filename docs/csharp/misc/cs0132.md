# 编译器错误 CS0132
“constructor”: 静态构造函数必须无参数  
  
 不能使用一个或多个参数声明[静态](../../visual-basic/language-reference/modifiers/static.md)构造函数。 有关详细信息，请参阅[构造函数](../../csharp/programming-guide/classes-and-structs/constructors.md)。  
  
 下面的示例生成 CS0132：  
  
```  
// CS0132.cs namespace MyNamespace { public class MyClass { public MyClass(int i) { } } public class MyClass2 : MyClass { static MyClass2(int i)   // CS0132 { } } }  
```
