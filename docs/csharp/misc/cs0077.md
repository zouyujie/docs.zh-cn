# 编译器错误 CS0077
as 运算符必须与引用类型或可以为 null 的类型一起使用（“int”是不可以为 null 值的类型）。  
  
 向 [as](../../csharp/language-reference/keywords/as.md) 运算符传递了[值类型](../../csharp/language-reference/keywords/value-types.md)。 因为 `as` 可以返回 [null](../../csharp/language-reference/keywords/null.md)，所以只能向其传递[引用类型](../../csharp/language-reference/keywords/reference-types.md)或可以为 null 的类型。 有关可以为 null 的类型的详细信息，请参见 [可以为 null 的类型](../../visual-basic/reference/command-line-compiler/index.md)。  
  
 下面的示例生成 CS0077：  
  
```  
// CS0077.cs using System; class C { } struct S { } class M { public static void Main() { object o1, o2; C c; S s; o1 = new C(); o2 = new S(); s = o2 as S;  // CS0077, S is not a reference type. // try the following line instead // c = o1 as C; } }  
```
