# 编译器错误 CS1671
命名空间声明不能有修饰符或特性  
  
 修饰符在应用于命名空间时没有意义，因此不允许使用修饰符。  
  
 下面的示例生成 CS1671：  
  
```  
// CS1671.cs public namespace NS // CS1671 { }  
```
