# 编译器错误 CS0008
从文件“file”读取元数据时发生错误 \-\-“description”  
  
 已为检索元数据成功打开 DLL，但它已损坏，因此无法从中读取数据。 有关更多信息，请参见[\/reference \(Import Metadata\)](../../csharp/language-reference/compiler-options/reference-compiler-option.md)。
