# 编译器警告（等级 1）CS3026
符合 CLS 的字段“field”不能是可变字段  
  
 可变变量不应符合 CLS。  
  
## 示例  
 下面的示例生成 CS3026。  
  
```  
// CS3026.cs [assembly:System.CLSCompliant(true)] public class Test { public volatile int v0 =0;   // CS3026 // To resolve remove the CLS-CComplient attribute. public static void Main() { } }  
  
```
