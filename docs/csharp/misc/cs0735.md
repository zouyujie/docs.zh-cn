# 编译器错误 CS0735
指定为 TypeForwardedTo 特性的参数的类型无效  
  
 以下示例生成 CS0735。  
  
```  
// CS735.cs // compile with: /target:library using System.Runtime.CompilerServices; [assembly:TypeForwardedTo(typeof(int[]))]   // CS0735 [assembly:TypeForwardedTo(typeof(string))]   // OK  
```
