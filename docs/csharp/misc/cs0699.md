# 编译器错误 CS0699
“generic”未定义类型参数“identifier”  
  
 类型参数在泛型定义中使用，但该泛型定义未在该泛型的类型参数的声明列表中找到。 如果类型参数使用的名称不一致，此情况也可能发生。  
  
 以下示例生成 CS0699：  
  
```  
// CS0699.cs class C<T> where U : I   // CS0699 – U is not a valid type parameter { }  
```
