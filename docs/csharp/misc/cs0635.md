# 编译器错误 CS0635
“attribute”：System.Interop.UnmanagedType.CustomMarshaller 需要命名参数 ComType 和Marshal。  
  
 当封送格式为 **System.Interop.UnmanagedType.CustomMarshaller** 时，必须指定 **ComType** 和 **Marshal** 参数。
