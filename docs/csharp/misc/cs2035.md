# 编译器错误 CS2035
命令行语法错误：“compiler\_option”选项缺少“:\<number\>”  
  
 某些编译器选项需要一个值。  
  
## 示例  
 下面的示例生成 CS2035。  
  
```  
// CS2035.cs // compile with: /baseaddress // CS2035 expected  
```
