# 编译器错误 CS1646
原意说明符 @ 后面应是关键字、标识符或字符串。  
  
 有关原意说明符“@”的用法，请参阅字符串参数。 仅允许原意说明符位于字符串、关键字或标识符之前。 若要解决此错误，从任何不适当的位置删除 @ 符号，或添加需要的字符串、关键字或标识符。  
  
 下面的示例生成 CS1646：  
  
```  
// CS1646 class C { int i = @5;  // CS1646 // Try this line instead: // int i = 5; }  
```
