# 编译器错误 CS0468
类型“type1”类型“type2”间具有多义性  
  
 当正在编译的程序集中的两种类型具有相同的完全限定名时，会生成此错误。 如果这两种类型都在添加的模块中，或者一种在添加的模块中，另一个在源中，也可能出现此错误。
