# 编译器错误 CS0139
没有要中断或继续的封闭循环  
  
 在循环外遇到了一个 break 或 continue 语句。  
  
 有关详细信息，请参阅[跳转语句](../../csharp/language-reference/keywords/jump-statements.md)。  
  
 下面的示例生成 CS0139 两次：  
  
```  
// CS0139.cs namespace x { public class a { public static void Main() { continue;  // CS0139 break;     // CS0139 } } }  
```
