# <a name="compiler-error-cs1559"></a>编译器错误 CS1559
无法对 Main 方法使用“object”，因为它是被导入的  
  
 无效的类指定给[/main](../../csharp/language-reference/compiler-options/main-compiler-option.md)编译器选项; 类不能用作一个位置[Main](../../csharp/programming-guide/main-and-command-args/index.md)方法。
