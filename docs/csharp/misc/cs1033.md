# 编译器错误 CS1033
源文件已超过在 PDB 中可表示的 16,707,565 行的限制；调试信息将不正确  
  
 源代码文件超出了编译器可以处理的最大允许行数。 若要解决此错误，请从原始文件创建两个或多个源代码文件。 最大行数为 268,435,454 行。 如果你在使用 **\/debug**，则使用超过 16,707,566 次将导致调试信息损坏。
