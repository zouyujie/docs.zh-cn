# 编译器错误 CS0715
“静态类”: 静态类不能包含用户定义的运算符  
  
 用户定义的运算符在类的实例上运行。 无法实例化静态类，因此无法创建供运算符执行操作的实例。 因此，用户定义的运算符不可用于静态类。  
  
 下面的示例生成 CS0715：  
  
```  
// CS0715.cs public static class C { public static C operator+(C c)  // CS0715 { } public static void Main() { } }  
```
