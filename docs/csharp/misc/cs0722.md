# 编译器错误 CS0722
“type”: 静态类型不能用作返回类型  
  
 静态类型用作返回类型时没有意义，因为无法创建静态类型的实例。  
  
 下面的示例生成 CS0722：  
  
```  
// CS0722.cs public static class SC { } public class CMain { public SC F()  // CS0722 { return null; } public static void Main() { } }  
```
