# &lt;example&gt;（C# 编程指南）
## 语法  
  
```  
<example>description</example>  
```  
  
#### 参数  
 `description`  
 代码示例的说明。  
  
## 备注  
 使用 \<example\> 标记可以指定使用方法或其他库成员的示例。  这通常涉及使用 [\<code\>](../../../csharp/programming-guide/xmldoc/code.md) 标记。  
  
 使用 [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) 进行编译可以将文档注释处理到文件中。
