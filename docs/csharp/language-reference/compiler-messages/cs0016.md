# Compiler Error CS0016
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

未能写入输出文件“\<文件\>”，原因是“\<原因\>”  
  
 编译器未能写入输出文件。  检查文件的路径，确保文件存在。  如果该位置已经存在一个以前生成的文件，请确保该文件可写入，并且当前没有进程将该文件锁定。  例如，确保在尝试生成时可执行文件没有运行。
