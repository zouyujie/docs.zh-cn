# Compiler Error CS0001
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

内部编译器错误  
  
 尝试确定编译器是否由于无法分析意外的语法而失败。  如果重复收到此错误，请与 Microsoft 联系。
