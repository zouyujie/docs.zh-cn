# using（C# 参考）
`using` 关键字有两个主要用途：  
  
-   作为指令，用于为命名空间创建别名或导入其他命名空间中定义的类型。  请参见 [using 指令](../../../csharp/language-reference/keywords/using-directive.md)。  
  
-   作为语句，用于定义一个范围，在此范围的末尾将释放对象。  请参见 [using 语句](../../../csharp/language-reference/keywords/using-statement.md)。
